# Minipass Email Signature - Ready to Use

## How to Use

1. Copy the HTML code below
2. Go to [htmlemail.io/inline](https://htmlemail.io/inline/)
3. Paste the HTML and click "Inline"
4. Copy the rendered result (Ctrl+A, Ctrl+C)
5. Paste into Gmail Settings → Signature
