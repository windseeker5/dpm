Mobile Behavior
  • Sidebar Toggle: Hamburger menu shows/hides sidebar with slide animation
  • Bottom Navigation: Fixed position with elevated FAB for primary action
  • Search: Collapses on mobile, expands when focused
  • Notifications: Hidden on mobile to save space
  • Content Padding: Reduces to 16px with bottom spacing for nav
