Troubleshooting Guide

If alerts are not displaying correctly:

  • Check that you're logged in as admin on the main site
  • Try hard refresh (Ctrl+F5 / Cmd+Shift+R)
  • Open browser developer tools and check console for errors
  • Verify Tabler CSS files are loading in the Network tab
  • This test page should work independent of authentication
