Explanation of the Fix:

Problem:

The .img-responsive-21x9 class uses padding-top: 42.8571428571% to maintain a 21:9 aspect ratio. When we used display: flex directly on this container, the icon was not properly positioned within the aspect-ratio space.

Solution:

We changed the styling to:

  1. Set position: relative on the .img-responsive container
  2. Create an absolutely positioned child div that covers the entire container (position: absolute; top: 0; left: 0; right: 0; bottom: 0;)
  3. Apply display: flex; align-items: center; justify-content: center; to this inner div to center the icon

Result:

The calendar icon is now perfectly centered within the aspect-ratio container, and the card maintains the same height as cards with actual images.
